### Use Anchor Residential Proxy

```bash
curl --request POST \
  --url https://api.anchorbrowser.io/v1/sessions \
  --header 'Content-Type: application/json' \
  --header 'anchor-api-key: <api-key>' \
  --data '{
  "session": {
    "proxy": {
      "type": "anchor_residential",
      "active": true
    }
  }
}'
```

### Use a Custom Proxy


```bash
curl --request POST \
  --url https://api.anchorbrowser.io/v1/sessions \
  --header 'Content-Type: application/json' \
  --header 'anchor-api-key: <api-key>' \
  --data '{
  "session": {
    "proxy": {
      "type": "custom",
      "server": "proxy.example.com",
      "username": "myUser",
      "password": "myPassword",
      "active": true
    }
  }
}'
```
