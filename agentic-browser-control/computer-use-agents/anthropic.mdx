Anthropic's Claude models with Computer Use capabilities enable AI agents to interact with web browsers through screenshots and native computer control actions.

## Overview

Anthropic Computer Use (`anthropic-cua`) leverages Claude's vision and reasoning capabilities to:
- Analyze screenshots of web pages
- Identify interactive elements
- Execute precise mouse and keyboard actions
- Complete complex multi-step workflows

## Supported Models

| Model | Model ID | Context Window | Best For |
|-------|----------|----------------|----------|
| Claude Opus 4.5 | `claude-opus-4-5` | 200K tokens | Most advanced reasoning, complex workflows |
| Claude Sonnet 4.5 | `claude-sonnet-4-5` | 200K tokens | Balanced performance, general automation |
| Claude Opus 4 | `claude-opus-4` | 200K tokens | Advanced coding and agentic tasks |
| Claude Sonnet 4 | `claude-sonnet-4` | 200K tokens | Fast, reliable automation |

## Code Example

<CodeGroup>

```javascript node.js
import Anchorbrowser from 'anchorbrowser';

const anchorClient = new Anchorbrowser({
  apiKey: process.env.ANCHORBROWSER_API_KEY
});

const response = await anchorClient.agent.task(
  'Navigate to GitHub and star the anchorbrowser repository',
  {
    taskOptions: {
      url: 'https://github.com',
      agent: 'anthropic-cua',
      model: 'claude-sonnet-4-5',
      maxSteps: 30,
      secretValues: {
        GITHUB_USERNAME: process.env.GITHUB_USERNAME,
        GITHUB_PASSWORD: process.env.GITHUB_PASSWORD
      }
    }
  }
);

console.log(response);
```

```python python
from anchorbrowser import Anchorbrowser
import os

anchor_client = Anchorbrowser(api_key=os.environ.get("ANCHORBROWSER_API_KEY"))

response = anchor_client.agent.task(
    'Navigate to GitHub and star the anchorbrowser repository',
    task_options={
        'url': 'https://github.com',
        'agent': 'anthropic-cua',
        'model': 'claude-sonnet-4-5',
        'max_steps': 30,
        'secret_values': {
            'GITHUB_USERNAME': os.environ.get('GITHUB_USERNAME'),
            'GITHUB_PASSWORD': os.environ.get('GITHUB_PASSWORD')
        }
    }
)

print(response)
```

</CodeGroup>

## Configuration Options

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent` | string | Must be `anthropic-cua` |
| `model` | string | Claude model to use (see supported models above) |
| `url` | string | Starting URL for the task |
| `max_steps` | integer | Maximum actions the agent can take (default: 40) |
| `secret_values` | object | Secure credentials passed to the agent |
| `output_schema` | object | JSON Schema for structured output |

## Best Practices

- **Use Sonnet 4.5** for most automation tasks - it offers the best balance of speed and accuracy
- **Use Opus 4.5** for complex multi-step workflows requiring advanced reasoning
- **Provide clear, specific prompts** describing the exact task to complete
- **Use secret_values** for any credentials - never include them in the prompt
