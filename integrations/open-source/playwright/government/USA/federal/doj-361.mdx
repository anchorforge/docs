# How to Automate DOJ Form 361 with Playwright

Automate bankruptcy filing fee installment payment request workflows with Playwright when APIs aren't available or sufficient. You'll eliminate manual payment arrangement processing and reduce case filing delays by automating repetitive fee waiver and installment processes. Use Playwright to interact with PACER and U.S. Trustee systems programmatically.

[View PACER developer resources](https://pacer.uscourts.gov/file-case/developer-resources) for available APIs when applicable.

## Setup

Install Playwright and configure authentication:
```bash
npm install playwright
```

## Automate Workflows

Create scripts for common DOJ Form 361 tasks:

```JavaScript
import { chromium } from 'playwright';

const browser = await chromium.launch();
const page = await browser.newPage();

// Navigate to PACER bankruptcy system
await page.goto('https://ecf.pacer.uscourts.gov/');
await page.fill('[data-testid="username"]', process.env.PACER_USERNAME);
await page.fill('[data-testid="password"]', process.env.PACER_PASSWORD);
await page.click('[data-testid="login-button"]');

// Start new installment payment application
await page.click('[data-testid="bankruptcy-filing"]');
await page.click('[data-testid="form-361"]');
await page.selectOption('[name="bankruptcy_chapter"]', 'chapter_7');

// Debtor information
await page.fill('[name="debtor_first_name"]', 'Michael');
await page.fill('[name="debtor_middle_name"]', 'James');
await page.fill('[name="debtor_last_name"]', 'Thompson');
await page.fill('[name="ssn"]', '456-78-9012');
await page.fill('[name="case_number"]', '24-12345');

// Court information
await page.selectOption('[name="district"]', 'northern_california');
await page.selectOption('[name="division"]', 'san_francisco');
await page.fill('[name="judge_name"]', 'Hon. Patricia Chen');

// Joint debtor (if applicable)
await page.check('[name="joint_petition"]');
await page.fill('[name="joint_first_name"]', 'Lisa');
await page.fill('[name="joint_middle_name"]', 'Marie');

// Financial situation
await page.fill('[name="monthly_income"]', '1850');

// Income sources
await page.fill('[name="employment_income"]', '1600');
await page.fill('[name="spouse_income"]', '250');
await page.fill('[name="other_income"]', '0');
await page.fill('[name="total_household_income"]', '1850');

// Expense breakdown

// Download confirmation
await page.click('[data-testid="download-confirmation"]');

await browser.close();
```
