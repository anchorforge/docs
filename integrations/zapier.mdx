## Quick start

Use our Zapier integration to enhance your automation workflows with Anchor Browser's powerful browser automation tools.

### Connect Anchor Browser

1. Search for "Anchor Browser" in the Zapier app directory
2. Click **Connect** and enter your Anchor Browser API key
3. Test the connection to verify your credentials

### Create a Zap

1. Create a new Zap and select your trigger app
2. Add Anchor Browser as an action step
3. Choose from available actions:
   - **Perform Web Task** - Execute AI-powered browser automation tasks
   - **Create Browser Session** - Start a new browser session
   - **Get Session Status** - Check the status of an existing session

### Configure your task

Use our [AI Task Completion](/agentic-browser-control/ai-task-completion) action to perform various browser automation tasks in a single step.

Example configuration:
- **Task prompt**: Describe what you want the browser to do
- **Start URL**: The webpage to start from
- **Session timeout**: How long the session should remain active
