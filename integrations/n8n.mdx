## About n8n

[n8n](https://n8n.io/) is an open-source workflow automation platform that lets you connect apps and services together to build powerful workflows â€” without writing code.

## Quick Start
Use our node to enhance your automation workflows with Anchor Browser powerful tools.

### Node Installation

Install the Anchor Browser node from the n8n community nodes:

```bash
npm install n8n-nodes-anchorbrowser
```
